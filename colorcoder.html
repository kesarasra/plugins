# ColorCoder — simple HTML/CSS/JS app This single-file project includes a Node.js static server and a single-page frontend that uses Bootstrap and Tailwind (CDNs) to provide a GUI color picker, complementary color, shades, conversions (HEX/RGB/HSL), grayscale shades, and recommended font-weight stack for headings and paragraph. --- package.json --- ```json { "name": "colorcoder", "version": "1.0.0", "description": "Simple color utility app (ColorCoder)", "main": "server.js", "scripts": { "start": "node server.js" }, "dependencies": { "express": "^4.18.2" } } ``` --- server.js --- ```js const express = require('express'); const path = require('path'); const app = express(); const PORT = process.env.PORT || 3000; app.use(express.static(path.join(__dirname, 'public'))); app.get('/', (req, res) => res.sendFile(path.join(__dirname, 'public', 'index.html'))); app.listen(PORT, () => console.log(`ColorCoder running at http://localhost:${PORT}`)); ``` --- public/index.html --- ```html ColorCoder
